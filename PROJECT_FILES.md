## What Needs to Be Done Next

### 1. Install Dependencies

#### Backend
```bash
cd backend
npm install
```
This will install:
- express
- mongoose
- cors
- dotenv
- multer
- pdf-parse
- express-validator
- nodemon (dev)

#### Frontend
```bash
cd frontend
npm install
```
This will install:
- react
- react-dom
- react-scripts
- axios
- react-router-dom
- chart.js
- react-chartjs-2
- react-icons

### 2. Setup MongoDB
- Install MongoDB if not already installed
- Start MongoDB service
- Database will be auto-created on first run

### 3. Run the Application

#### Terminal 1 - Backend
```bash
cd backend
npm run dev
```

#### Terminal 2 - Frontend
```bash
cd frontend
npm start
```
